OCR Book

Reconstrucción inteligente de libros escaneados, con traducción automática opcional.

Convierte PDFs escaneados (o imágenes sueltas) en documentos digitales limpios y agradables de leer: reconstruye el orden de lectura, los párrafos, los títulos, las listas, las citas y las notas al pie como lo haría un libro nacido digital. Exporta a PDF, DOCX, HTML, EPUB, TXT y Markdown.

Motores OCR intercambiables

Tesseract por defecto; PaddleOCR y EasyOCR como alternativas opcionales.

Traducción consciente de estructura

Traduce a nivel de párrafo/título, no palabra por palabra, con Claude o de forma local/offline.

CLI, GUI y web

Línea de comandos para lotes, interfaz de escritorio (PySide6) e interfaz web (FastAPI).

Esta página es solo una landing estática

GitHub Pages solo puede servir archivos estáticos: no ejecuta Python ni el pipeline de OCR. Para usar la interfaz web hace falta levantar el servidor (ocrbook-web) en tu propia máquina o en un servidor propio. Si ya tenés uno desplegado, pegá su URL acá abajo para ir directo al login:

Instrucciones para desplegarlo

Empezar en local

pip install -e ".[web]"
export OCRBOOK_WEB_TOKEN="elegí-un-token-largo-y-aleatorio"
ocrbook-web

Ver el README del proyecto para la instalación completa (CLI, GUI e interfaz web) y los requisitos del sistema.
